# 宠物诊所管理系统 java1068

## 项目概述

宠物诊所管理系统是一个基于Spring、mybatis和vue技术的管理系统。它涵盖了首页管理、公告信息、宠物信息、管理员信息、医生信息、柜员信息、药品信息、药品领取以及个人信息等多个功能模块。

## 技术栈

- **架构模式：** MVC/前后端分离
- **后端技术：**
  - JDK版本：Java JDK 1.8
  - 服务器：SpringBoot（内嵌Tomcat）
- **前端技术：**
  - node版本：14.16.1
  - 开发框架：vue
- **数据库：**
  - 数据库版本：mysql8.0
  - 数据库可视化工具：navicat

## 功能模块

- 首页管理
- 公告信息管理
- 宠物信息管理
- 管理员信息管理
- 医生信息管理
- 柜员信息管理
- 药品信息管理
- 药品领取管理
- 个人信息管理

## 系统角色

- 系统管理员
- 医生
- 柜员

## 运行环境

- 后端：Java JDK 1.8 或更高版本，Mysql 8.0 或更高版本
- 前端：node.js 14.16.1 或更高版本

## 部署步骤

1. 准备运行环境：安装Java JDK 1.8、Mysql 8.0和node.js 14.16.1。
2. 导入数据库结构。
3. 部署后端服务：通过IDEA或其他工具构建并运行SpringBoot应用程序。
4. 部署前端服务：在项目目录下运行npm安装依赖，然后运行npm run dev或相应的命令启动前端服务。
